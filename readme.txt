Displays 'n' recent posts, including custom post typea,s as multiple rows of 2 thumbnails each, in a widget. 

== Description ==
This plugin will provide a visual, easy to use recent custom post types widget. 
Permits you to show a list of the latest custom post types with thumbnail and post date.

== Features ==
* Two thumbnails per row
* Custom post type included
* Customize Widget Title
* Customize number of posts
* Customize number of posts to offset
* Show/Hide title of posts 
* Show/Hide date of posts


== Installation ==
**Automatic installation**
To do an automatic installation of Rapo Recent Custom Posts Widget .

1. Log in to your WordPress dashboard and go to Plugins -> Add New
2. In the search field type **Rapo Recent Custom Posts Widget** and click Search Plugins.
3. Find **Rapo Recent Custom Posts Widget** and **click on Install Now**
4. Activate the plugin.
5. Go to the widgets page **Appearance -> Widgets**.
6. Find **Rapo Recent Custom Posts Widget** widget.

**Manual installation or Installing Via FTP**

*The manual installation method involves downloading our plugin and uploading it to your webserver via your favorite FTP application.

1. Download the plugin to your hardisk.
2. Unzip.
3. Upload the **rapo-recent-custom-posts-widget** folder into your plugins directory.
4. Log in to your WordPress admin panel and click on Plugins.
5. Then activate the plugin.
6. Go to the widgets page **Appearance -> Widgets**.
7. Find  **Rapo Recent Custom Posts Widget** widget.
